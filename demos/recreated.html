Static/Recreated UNav

The UNav above has been recreated from the original embedded version. The UNav should only be recreated when embedding the original script isn't possible on a platform, e.g inline JavaScript/CSS is restricted, external resources are not allowed for security reasons.

⚠️ Please request an exception from the NYS Design System team via our UNav Intake Form before recreating the UNav.

When recreating, be sure to include the necessary CSS and asset files directly within your platform. Do not hotlink to external sources, as those references will likely break. For a seamless integration:

  • Download and host the fonts, icons, and image assets (located in /demos/assets/*) locally within your environment.
  • Ensure that all paths to fonts, CSS, and javascript files are correctly updated to point to your hosted assets.
  • Test thoroughly within your application setup to confirm that the recreated UNav functions correctly across all supported browsers and devices.
